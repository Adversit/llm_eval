# 文档问答生成管理系统

这是一个基于Streamlit构建的文档处理管理仪表盘，用于批量处理文档文件并生成问答对。

## 功能特点

- 支持多文件上传与批量处理
- 可视化处理状态与进度
- 实时查看处理结果
- 预览和下载生成的Excel文件
- 灵活配置处理参数

## 使用方法

1. 运行启动脚本：

```bash
python run_dashboard.py
```

2. 在浏览器中打开显示的URL（通常是 http://localhost:8501）

3. 使用界面：
   - 左侧边栏：配置处理参数
   - 文件上传：选择并上传Word文档文件
   - 处理状态：查看文件处理进度
   - 结果查看：浏览和下载生成的文件

## 参数说明

- **每段内容生成问答对数量**：设置从每个内容段落生成多少个问答对
- **请模型建议问答对数量**：是否让模型根据内容建议适合的问答对数量
- **使用模型建议的问答对数量**：是否采用模型建议的问答对数目
- **最低事实依据分数**：设置问答对事实准确性的最低评分阈值（1-10）
- **最低总体质量分数**：设置问答对整体质量的最低评分阈值（1-10）
- **评估样本百分比**：设置要评估的问答对样本比例，以节省处理时间
- **包含评估理由**：是否在结果中包含详细的评估理由
- **跳过步骤选项**：可选择跳过特定处理步骤以加快处理速度

## 系统要求

- Python 3.7+
- Streamlit
- Pandas
- 项目其他依赖项

## 注意事项

- 大文件处理可能需要较长时间
- 所有上传的文件会被临时存储并在处理完成后删除
